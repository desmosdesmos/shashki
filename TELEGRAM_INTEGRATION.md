# Интеграция с Telegram Bot API

Для запуска этого Telegram Mini App в Telegram, вам нужно выполнить несколько шагов:

## 1. Создание бота

1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания нового бота
4. Сохраните токен, который вы получите

## 2. Настройка веб-приложения

1. После создания бота, отправьте команду `/newapp`
2. Загрузите изображение-иконку для вашего mini app
3. Введите название вашего приложения
4. В поле URL введите адрес, по которому будет размещено ваше приложение (например, `https://yourdomain.com`)

## 3. Настройка сервера

1. Установите переменную окружения `TELEGRAM_BOT_TOKEN` в файле `.env` в папке `backend`
2. Убедитесь, что ваш сервер доступен по HTTPS (необходимо для Telegram Mini Apps)
3. Запустите сервер командой `npm run dev`

## 4. Особенности разработки

- Все взаимодействие с пользователем происходит внутри Telegram
- Для получения информации о пользователе используйте Telegram Web Apps SDK
- Не храните конфиденциальные данные на клиенте
- Используйте методы SDK для взаимодействия с Telegram (отправка данных, закрытие приложения и т.д.)

## 5. Полезные ресурсы

- [Официальная документация Telegram Bot API](https://core.telegram.org/bots/api)
- [Руководство по Telegram Mini Apps](https://core.telegram.org/bots/webapps)
- [Telegram Web Apps SDK](https://github.com/Telegram-Mini-Apps/telegram-web-apps)

## 6. Тестирование

Для тестирования приложения в Telegram:

1. Отправьте своему боту команду, которая открывает mini app (например, кнопку или сообщение с ссылкой)
2. Убедитесь, что все функции работают корректно
3. Проверьте работу на разных устройствах и размерах экрана
