# Telegram Checkers Mini App

Это Telegram Mini App для игры в шашки. Игроки могут играть как против ИИ, так и против других игроков онлайн.

## Особенности

- Игра в шашки с красивым интерфейсом
- Режим игры против ИИ с несколькими уровнями сложности
- Режим онлайн-игры с другим игроком
- Интеграция с Telegram
- Возможность пригласить друзей по ссылке
- Адаптивный дизайн для мобильных устройств

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Установите зависимости для бэкенда:
```bash
cd backend
npm install
```

3. Установите зависимости для фронтенда:
```bash
cd frontend
npm install
```

4. Создайте файл `.env` в папке `backend` и укажите:
```
NODE_ENV=development
PORT=5000
MONGODB_URI=mongodb://localhost:27017/checkers
TELEGRAM_BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN_HERE
```

5. Запустите приложение:
```bash
npm run dev
```

## API

### Игровые маршруты

- `POST /api/games` - Создать новую игру
- `GET /api/games/:id` - Получить информацию об игре
- `PUT /api/games/:id` - Сделать ход

### Пользовательские маршруты

- `POST /api/users` - Создать пользователя
- `GET /api/users/:id` - Получить информацию о пользователе

## Интеграция с Telegram

Приложение использует Telegram Web Apps для интеграции с Telegram. Для запуска в Telegram необходимо:

1. Создать бота через @BotFather
2. Получить токен бота
3. Указать токен в переменной окружения `TELEGRAM_BOT_TOKEN`
4. Настроить веб-приложение через @BotFather

## Правила игры

- Шашки ходят по диагонали на одну клетку вперед
- Дамки могут ходить на любое количество клеток по диагонали
- Взятие обязательно - если есть возможность побить фигуру противника, нужно это сделать
- При достижении последней линии шашка становится дамкой
